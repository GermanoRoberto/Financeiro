# Roadmap do Projeto

## ✅ MVP (Fase 1) - CONCLUÍDO

- [x] Setup básico Next.js 14 + Tailwind
- [x] Integração Supabase (Auth + Database)
- [x] Schema SQL completo com RLS
- [x] Login via Google OAuth
- [x] Extração de contracheques via Gemini API
- [x] Tela de conferência/edição
- [x] Cadastro manual de dívidas
- [x] Dashboard com visão Casal/Individual
- [x] Cálculo de comprometimento de renda
- [x] Gráficos e projeção mensal
- [x] Bot Telegram básico
- [x] Webhook Telegram
